javascript之正则表达式

发表于 2018-05-11 | 分类于 JS

正则表达式是一个描述字符模式的对象。正则表达式可以进行强大的模式匹配和文本检索与替换功能 (md partial supported)

一直都想把正则再学习一次，一直都没有实践，最近终于下定决心再一次把整理学习下。

学习之前推荐一个网站可以形象的显示正则的规则，对于我们理解非常有帮助

正则好帮手：http://regexper.com

具体详细可以看自己的笔记，在这里只是记录一些学习体会和常见的语法

  1. g :global 全文搜索 不添加 搜索到第一个匹配停止

    i :ignore case 忽略大小写 默认大小写敏感

    m :multiple lines 多行搜搜

  2. 使用元字符^ 在[]里面使用标示反向类；[^abc]表示不是字符a或b或c的内容

  3. 匹配横线

  4. 边界：
    ^以xxx开始；
    $以xxx结束； \b单词边界； \B非单词边界

  5. 量词默认是贪婪模式匹配

  6. 量词只会匹配到和它紧挨着的，如果需要扩大范围那么需要加上括号进行分组

  1. 前瞻

  2. lastIndex
    表示当前表达式匹配的内容的最后一个字符的下一个位置

  3. exec(str) 方法

  4. search(reg) 方法
    方法返回第一个匹配结果index,查找不到返回-1
    会忽略g 全局匹配

  5. match(reg)方法：
    在全局和非全局下返回的值有区别
    非全局：返回一个数组，里面有与他匹配相关的信息如index input
    全局：返回的是数组，元素是所有匹配的子串，无index input属性
    12.split(reg)方法

  6. replace(str,replaceStr)

  7. replace(str,function)
    function有四个参数：
    1：匹配的字符串
    2：z正则表达式的分组内容，没有分组则没有该参数
    3：匹配项在字符串中的index
    4:原字符串

将JAVA WEB项目在本地运行

发表于 2017-07-28 | 分类于 others

最近做的项目是一个java web项目，需要在本地把项目跑起来，数据库，接口什么的都在本地，这里总结了自己如何把它跑起来

如何打包成war

首先是环境的配置：
http://www.cnblogs.com/pannysp/archive/2012/03/07/2383364.html

配置好环境了一行，要看是不是maven项目，如果是的话，进入到项目包含有pom.xml的文件夹位置，然后运行代码：mvn clean && mvn compile && mvn package 就可以成功打包了，打包后的位置可以看cmd里面最后几句代码就能看到了

部署到tomcat里面

https://blog.csdn.net/zhanglf02/article/details/73565354

发表于 2017-03-28 | 分类于 HEXO

问题总结：

  1. localhost:4000打不开，但是在bash里面显示已经running了
    我这里是因为端口占用了，使用命令hexo server -p 4001
    解决

  2. 不知道在文件的那个位置运行hexo的g s d ?
    我这里是在blog文件夹里面运行MINGW64 /d/hexo/blog

  3. 如何编辑首页的样式？
    D:\hexo\blog\themes\next\source\css_custom 编辑下面的custom.styl内容就可以了，直接写css

  4. 设置每篇文章的高度，不至于全部显示在首页
    进入hexo博客项目的themes/next目录
    用文本编辑器打开_config.yml文件
    搜索”auto_excerpt”,找到如下部分设置false为true

  5. 如何设置标签页？
    blog 目录下执行以下命令hexo new page “tags”
    编辑这个index.md，添加type: “tags”
    在md文章前面写上tags: [html, css, js]

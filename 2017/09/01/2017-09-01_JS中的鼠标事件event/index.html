JS中的鼠标事件event

发表于 2017-09-01 | 分类于 JS

这里主要分析下经常会遇到的一些问题：

  • click
  • dblclick
  • mouseup/mousedown 鼠标弹起和按下
  • contextmenu 鼠标右键点击（在右键菜单显示前）
  • mouseover/mouseout 鼠标进入和离开（要冒泡）
  • mouseenter/mouseleave 鼠标进入和离开
  • mousemove 鼠标移动

target和currentTarget的区别：

  • event.currentTarget返回绑定事件的元素（这里一直指out）
  • event.target返回触发事件的元素（这里指触发mouseover的元素，会变化）

==例子1：==

鼠标事件属性event

click和dblclick事件：

click:单击事件，事件的触发情况：

  • 先触发mousedown
  • 再触发mouseup
  • 最后触发click事件

dblclick：双击事件，事件的触发情况：

  • 触发mousedown
  • 触发mouseup
  • 触发click
  • 触发mousedown
  • 触发mouseup
  • 触发click
  • 最后才触发dblclick

contextmenu 鼠标右键点击事件

这个事件是在右键菜单弹出来之前触发的，所以我们可以在这个事件里面调用e.preventDefault();来阻止右键菜单的弹出，然后自定义一个右键菜单，这个情况在很多后台管理系统的设计上面会使用。 另外键盘空格右边第三个键也会触发这个事件！

mouseenter/mouseleave mouseover/mouseout：

mouseover/mouseout：这两个事件会生冒泡。具体情况比如：在父级元素绑定mouseover事件，那么从父级元素进入子元素的时候会触发mouseover事件（这个mouseover是由子元素触发的），然后这个mouseover事件会冒泡到父级，然后父级元素执行事件处理函数。
